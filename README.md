# Guía para Clonar el Repositorio Ingeweb1.0

## Opción 1: Clonar con HTTPS

Abre tu terminal o línea de comandos y ejecuta:
```bash
git clone https://github.com/273Manuel/Ingeweb1.0.git
```

Luego accede a la carpeta del proyecto:
```bash
cd Ingeweb1.0
```

## Opción 2: Clonar con SSH (si tienes configurada tu clave SSH)
```bash
git clone git@github.com:273Manuel/Ingeweb1.0.git
cd Ingeweb1.0
```

## Opción 3: Descargar como ZIP

Si no tienes Git instalado, puedes descargar el repositorio directamente:

1. Ve a https://github.com/273Manuel/Ingeweb1.0
2. Haz clic en el botón verde **"Code"**
3. Selecciona **"Download ZIP"**
4. Extrae el archivo en tu computadora

## Verificar el contenido

Una vez clonado, puedes ver los archivos con:
```bash
ls
```

Deberías ver:
- `guia1ingweb/` (carpeta con el proyecto)
- `README.md` (documentación del proyecto)

## Abrir el proyecto

Para ver la página web, navega a la carpeta y abre el archivo `index.html` en tu navegador preferido.
